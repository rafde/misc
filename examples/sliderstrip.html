Responsive Sliderstrip example

This page is an example of how a responsive sliderstrip/carousel could work. There are times when the client wants it to appear under certain viewport widths cases. This uses the following:

  • CSS for layout and animation
  • jQuery and jQuery mobile (for touch events)
  • Ctrl (custom code I produced to trim down boilerplate code and abstraction)
  • SliderStrip (extends Ctrl)
  • PriorityEventBroadcaster (augmented Publish/Subscribe JS Pattern)

Note: Please do not request to use this code. I am still deviating a lot of the implementation for better performance.
